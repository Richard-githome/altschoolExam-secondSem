- Node.js (version 18.13.0)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Richard-githome/altschoolExam-secondSem
```

2. Install dependencies:

```bash
cd altschoolExam-secondSem
npm install
```

3. Start the server:

```bash
npm run dev
```
That is all.
